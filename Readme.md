# Para usar Streamlit

### se instala con lo siguiente:

### crear entorno virtual con

### python -m venv .venv

## activamos el entorno con el siguiente comando

# Windows command prompt

.venv\Scripts\activate.bat

# Windows PowerShell

.venv\Scripts\Activate.ps1

# macOS and Linux

source .venv/bin/activate

## como instalar

## pip install streamlit

#### para usar la pagina demo streamlit Hello ejemplo app

##### streamlit hello

##### si no funciona

##### python -m streamlit hello

## para ejecutar una aplicacion que usa streamlit

## streamlit run .\stream.py

## Para este proyecto tenemos el archivo de requirements.txt

## para instalar pip install -r requirements.txt
